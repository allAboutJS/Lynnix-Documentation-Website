File-based routing · htmx-native

The missing backend for htmx applications

Lynnix is a file-based routing and SSR middleware for Node.js. Drop your files in the right place, export a function, and get routing, rendering, middleware chains, htmx-aware partial responses, and error boundaries — all wired together automatically.

Explore docs GitHub

Filesystem routing

Static, dynamic, and catch-all routes from your directory structure. No config required.

htmx-aware

Detects htmx requests automatically and serves fragment.html instead of page.html.

Framework-agnostic

Works as middleware with Express, Fastify, or bare node:http. Mount it anywhere.

server.js
Lynnix 
import { createLynnixApp } from "lynnix";
import express from "express";

const app = express();
const handler = await createLynnixApp("app");

app.use(express.static("public"));
app.use(handler);

app.listen(3000);
